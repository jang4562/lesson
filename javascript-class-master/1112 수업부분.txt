 서버 : 클라이언트가 접속하여 웹페이지를 요청하면, 데이터, 이미지, 동영상 등의 자료들과 HTML,CSS,JS 등으로 웹페에지를 구성하여 클라이언트에게 응답하는 역활  //

* 리소스 :  웹페이지 내부에 표시되는 데이터,이미지,동영상을 통칭하여 부르는 용어

ㄴ서버는 클라이언트 가 (웹브라우저) 자신에게 접속하기를 기다린다.
ㄴ사용자1은 www.naver.com/news 로 접속하고,사용자2는 www.naver.com/webtoon으로 접속한다
ㄴ서버는 주소를 확인하여 자신에게 접속한것이 맞는지 확인한다.
*메인도메인 (www.naver.com)을 확인하여 자신에게 접속하였음을 확인
*서브도메인(/news,/webtoon)을 확인하여 서버에서 지정한 서브도메인인지 확인한다
*서버에서 지정한 서브 도메인이 맞다면, 각 서브 도메인에 맞는 뉴스(/news),웹툰(/webtoon) 웹페이지를 구성한다.
